- 不可继承的：
    - display
    - margin
    - padding
    - border
    - background
    - height   min-height   max-height
    - width   min-width   max-width
    - overflow
    - position   left   right   top   bottom   z-index
    - float
    - vertical-align
- 所有元素可继承：
    - visibility 
    - cursor
- 内联元素可继承：
    - line-height
    - color
    - font   font-family   font-size   font-style   font-weight
- 块状元素可继承：
    - text-indent
    - text-align
- 列表元素可继承：
    - list-style
